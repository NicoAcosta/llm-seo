# LLM SEO - Codex Installation

1. Clone this repo:
   ```bash
   git clone https://github.com/NicoAcosta/llm-seo.git
   ```

2. Symlink the skill into your Codex skills directory:
   ```bash
   ln -s /path/to/llm-seo/skills/llm-seo ~/.codex/skills/llm-seo
   ```

3. Restart Codex to pick up the new skill.
